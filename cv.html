SOFTWARE APPLICATION PROJECTS

CYCHO | SOLO PROJECT

A passion project to address the growing number of London cyclists and scooter commuters and the tedious repair process. This is a full stack app, planned out with Canva, logos in Photoshop and built using JavaScript, MongoDB, Express and React and carefully styled with CSS. Building this tested my understanding and a chance to experiment with features and tools. This app includes an interactive map, displaying markers from the database, an image uploader and I have been playing around with a Watson Assistant chatbot made with redux.

GLASGOW MURAL TRAIL | GROUP PROJECT

This group project addressed the problem that the official Glasgow Mural Trail has encountered, their key feature map is not working due to changes in the Google Maps API. We agreed an interactive map is the most appealing way to display data to our personas so implemented one using leaflet.js and built a full stack using JavaScript, MongoDB, Express, and React framework. This had the functionality to search through murals with filter through any of their properties, display their locations or the map, like and hyperlink their Instagrams. We styled this with CSS, inspired by the Airbnb app and it turned out well, clear, simple, attractive.

FIT__INIT__ | SOLO PROJECT

An interpretation on our project briefs, I decided that a gym app was one that resonated with me most as I have a good understanding of how I’d want it to look and feel. This app was designed to be used by a gym manager to add, update and delete trainers to classes. This is a full stack interactive app, built using Python, Flask and React and styled with CSS.
