Glasgow Murals | Full Stack | Express/React

A javascript react app which aims to provide information on murals in Glasgow and promote a positive image of the city. We created a database of murals based on the Glasgow mural trail website, which are presented in a list and a Leaflet.js map in the front end. Users can filter the murals by title, artist name, year, location and description and the list and map will update instantly. Users can also like or unlike a mural.
